# 🚀 RISEROUTE – AI Career Navigator

RISEROUTE is a web platform that helps students choose a technology career path and provides a complete roadmap to achieve it.

Students can select from predefined tech roles such as **Software Engineer, AI/ML Engineer, Full Stack Developer, Data Analyst, Cloud Engineer, and Cybersecurity Specialist**.
After selecting a role, the platform provides curated learning resources, certifications, playlists, and job opportunities.

---

## 🎯 Features

* 🎓 Career role selection (6 tech roles)
* 📺 Best YouTube playlists for learning
* 📜 Recommended certifications
* 🗺️ Complete learning roadmap
* 💼 Job & internship application links
* 📊 Market demand and industry scope
